<h3>⚡ Featured Software & Data Projects</h3>

### 📊 [NLP-and-ML-Sentiment-Analysis-Prediction](https://github.com/SyedAliRazaGilani/NLP-and-ML-Sentiment-Analysis-Prediction)
End-to-end **NLP sentiment classification** on Amazon Alexa reviews: EDA + preprocessing (NLTK + bag-of-words), model training (RF / DT / XGBoost), and deployment via **Flask API + web UI**.

- 🔹 **Live demo**: [nlp-and-ml-sentiment-analysis-prediction](https://nlp-and-ml-sentiment-analysis-prediction-itleygupq.vercel.app/)
- 🔹 **Tech**: Python, scikit-learn, NLTK, XGBoost, Flask

### 🌦️ [Weather-Prediction-MachineLearningAlgorithms](https://github.com/SyedAliRazaGilani/Weather-Prediction-MachineLearningAlgorithms)
Benchmarked **SVR/SVM, XGBoost, and LSTM-family** time-series models for humidity forecasting using smart-city sensor data, comparing **Edge AI (TensorFlow Lite)** vs **cloud inference (GCP)**. Best reported centralized performance: \(R^2 = 0.8451\), **MAE = 0.0164**.

- 🔹 **Notebook**: [Open in Colab](https://colab.research.google.com/github/SyedAliRazaGilani/Weather-Prediction-MachineLearningAlgorithms/blob/main/MachineLearningModelsforWeatherPrediction.ipynb)
- 🔹 **Tech**: Python, TensorFlow, scikit-learn, XGBoost

### 🎯 [Train-and-Deploy-YOLO-Models](https://github.com/SyedAliRazaGilani/Train-and-Deploy-YOLO-Models)
Trained and deployed an **Ultralytics YOLO** object detector for candy types, then layered a nutrition lookup to estimate **calories/sugar** from live detections (useful for quick “what’s on the table” tallies).

- 🔹 **Notebook**: [Open in Colab](https://colab.research.google.com/github/SyedAliRazaGilani/Train-and-Deploy-YOLO-Models/blob/main/Train_YOLO_Models.ipynb)
- 🔹 **Tech**: Python, Ultralytics YOLO

### 🧠 [RAG-Pipeline](https://github.com/SyedAliRazaGilani/RAG-Pipeline)
A complete **RAG pipeline** for NASA knowledge: vector embeddings + Astra DB semantic search + streaming chat UI, with responses generated by **Google Gemini**.

- 🔹 **Live demo**: [NASA AI Space Lovers](https://rag-nasaai-spacelovers.onrender.com/)
- 🔹 **Tech**: Next.js, TypeScript, Astra DB (vector), Gemini

### 🎤 [Voice-to-Voice-LLM](https://github.com/SyedAliRazaGilani/Voice-to-Voice-LLM)
Voice assistant pipeline: **speech-to-text → Groq LLM → text-to-speech**, with a modern web UI and Flask backend.

- 🔹 **Live demo**: [voice-to-voice-llm.vercel.app](https://voice-to-voice-llm.vercel.app/)
- 🔹 **Tech**: React, Flask, Groq, SpeechRecognition, gTTS

### 💾 [Matrix-SpaceSaver](https://github.com/SyedAliRazaGilani/Matrix-SpaceSaver)
Sparse matrix calculator in **C++** that stores only non-zero values and supports operations like add/subtract, multiply, transpose, determinant, and inverse.

- 🔹 **Tech**: C++

### 📈 [DataAnalyserPro](https://github.com/SyedAliRazaGilani/DataAnalyserPro)
Privacy-first **in-browser** Excel/CSV data analyzer with instant stats + charts (no server uploads; everything runs client-side).

- 🔹 **Live demo**: [data-analyser-pro.netlify.app](https://data-analyser-pro.netlify.app/)
- 🔹 **Tech**: React, TypeScript, Vite, TailwindCSS, Recharts

### 📚 [FullStack-Mobile-Application-ReviewShelf](https://github.com/SyedAliRazaGilani/FullStack-Mobile-Application-ReviewShelf)
Full-stack **React Native** mobile app for sharing book reviews with JWT auth, infinite scrolling feed, post creation/deletion, and **Cloudinary** image uploads (Node/Express/Mongo backend).

- 🔹 **Tech**: React Native, Expo Router, Node.js, Express, MongoDB, Cloudinary
